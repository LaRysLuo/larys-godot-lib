# StringUtils 使用说明

> 提供一组常用的字符串处理静态函数，便于在游戏或工具中快速调用。

---


## 静态类方法

- **`random_string(length: int = 4)`**
  - **功能**：
  - 生成由大写英文字母和数字组成的随机字符串。
  - **参数**: 生成的长度数量，默认长度为 `4`
  - **返回值**：字符串


- **`is_blank(text: String)`**
  - **功能**：
    - 判断字符串是否为空或只包含空白字符（如空格、换行、制表符等）。
  - **参数**: 要判断的字符串
  - **返回值**：布尔值
